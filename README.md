# java-cli-sbt-jacoco-testng-hello-world

## Description
A POC for sbt app using testng and jacoco.
Writes test results to console
and html.

## Tech stack
- java
- sbt
  - testng
  - jacoco

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
